# AntiCaps Plugin

A lightweight Minecraft plugin that prevents excess capitalisation in chat messages. Should support MC version 1.8.8 to 1.21.11

## Commands

- `/anticaps reload` - Reload configuration

## Permissions

- `anticaps.reload` - Access the reload command (default: op)
- `anticaps.bypass` - Bypass the filter (default: op)

## Building from Source

1. Clone the repository
2. Ensure you have Maven installed
3. Run `mvn clean install`
4. The compiled JAR will be in the `target/` directory
